# 学术论文写作系统 UI 设计文档

## 1. 项目概述

学术论文写作系统是一个基于 AI 代理协作的综合平台，集成了文献检索、可行性分析、数据处理、论文撰写、格式优化和同行评审等功能。系统旨在大幅提升论文写作效率和质量，通过核心 MCP 组件、专业 Subagents 和完整的工作流程，为用户提供一站式的学术论文写作解决方案。

## 2. 技术栈

- **前端框架**：Vue 3
- **UI 组件库**：Element Plus
- **路由管理**：Vue Router 4
- **构建工具**：Vite
- **代码高亮**：highlight.js
- **Markdown 解析**：marked

## 3. 页面整体布局结构

## 6. 响应式设计要求

### 6.1 断点设置

| 断点 | 屏幕宽度 | 布局调整 |
|------|---------|---------|
| 桌面端 | > 768px | 三栏布局（顶部导航栏 + 侧边栏 + 主内容区） |
| 移动端 | ≤ 768px | 单列布局（顶部导航栏 + 主内容区，侧边栏折叠） |

### 6.2 布局调整

- **顶部导航栏**：在小屏幕上减小 logo 字体大小，缩短搜索框宽度
- **侧边栏**：在小屏幕上调整为宽度 100%，显示在主内容区上方
- **主内容区**：在小屏幕上调整内边距，优化内容布局
- **卡片布局**：在小屏幕上从多列调整为单列
- **步骤布局**：在小屏幕上调整为垂直排列

### 6.3 交互调整

- **菜单交互**：在小屏幕上调整为折叠菜单
- **按钮大小**：在小屏幕上适当调整按钮大小
- **字体大小**：在小屏幕上适当调整字体大小，确保可读性

## 7. 关键用户流程

### 7.1 系统配置流程

1. **访问系统配置页**：从侧边栏点击 "系统配置" 菜单项
2. **查看环境要求**：了解系统所需的软件环境
3. **安装依赖**：按照指引安装 Python 和 Node.js 依赖
4. **配置 MCP 服务器**：编辑配置文件，添加 MCP 服务器配置
5. **启动 MCP 服务器**：使用命令行启动 MCP 服务器
6. **查看 Subagents 配置**：了解系统内置的专业 Agent

### 7.2 文献调研流程

1. **访问首页**：从侧边栏点击 "首页" 菜单项
2. **了解核心功能**：查看系统的核心功能，特别是文献检索功能
3. **学习使用指南**：点击 "查看使用指南" 按钮，了解系统的使用方法
4. **开始文献调研**：使用系统的文献调研专家 Agent 进行文献检索和分析

### 7.3 论文撰写流程

1. **访问系统组件页**：了解系统的核心组件和工作流程
2. **配置系统**：按照系统配置指南配置系统
3. **开始撰写**：使用论文撰写专家 Agent 辅助撰写论文
4. **格式优化**：使用格式优化专家 Agent 调整论文格式
5. **同行评审**：使用同行评审专家 Agent 进行论文评审
6. **修改完善**：根据评审意见修改完善论文

### 7.4 搜索流程

1. **输入搜索关键词**：在顶部导航栏的搜索框中输入关键词
2. **执行搜索**：点击搜索按钮或按回车键执行搜索
3. **查看搜索结果**：在搜索页查看搜索结果
4. **过滤和排序**：根据需要过滤和排序搜索结果
5. **查看详情**：点击搜索结果查看详情
